# Archivos SolidWorks

Esta carpeta contiene los archivos originales de **SolidWorks** utilizados para el diseño mecánico del robot móvil **AMR-BC**.

Estos archivos corresponden a los modelos editables de las piezas diseñadas durante el desarrollo del Trabajo Fin de Grado, incluyendo soportes, elementos estructurales y componentes auxiliares integrados en la arquitectura mecánica del robot.

## Finalidad

Los archivos incluidos en esta carpeta permiten modificar, adaptar o reutilizar el diseño mecánico del robot en futuras versiones del proyecto.

A diferencia de los archivos `.stl`, que están orientados principalmente a la impresión 3D, los archivos de SolidWorks permiten editar la geometría original de las piezas.

## Contenido

Esta carpeta puede incluir archivos como:

| Formato | Descripción |
|---|---|
| `.sldprt` | Piezas individuales diseñadas en SolidWorks |
| `.sldasm` | Ensamblajes del robot o subconjuntos mecánicos |
| `.slddrw` | Planos técnicos asociados a las piezas, si están disponibles |

## Piezas diseñadas

Entre los elementos mecánicos diseñados se incluyen:

- Soportes para sensores.
- Soporte para el LiDAR RPLIDAR C1.
- Soportes para Raspberry Pi 5 y OpenRB-150.
- Soporte para el convertidor DC-DC LM2596.
- Elementos estructurales del chasis.
- Llantas y neumáticos diseñados para impresión 3D.
- Soportes auxiliares para la integración física del sistema.

## Nota

Para fabricar directamente las piezas mediante impresión 3D, se recomienda utilizar los archivos disponibles en la carpeta `cad/stl/`.

Los archivos de esta carpeta están pensados principalmente para edición, modificación y documentación del diseño mecánico original.
