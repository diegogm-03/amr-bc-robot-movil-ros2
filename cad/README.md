# Archivos CAD

Esta carpeta contiene los archivos relacionados con el diseño mecánico del robot móvil **AMR-BC**.

Las piezas fueron diseñadas mediante software CAD y fabricadas mediante impresión 3D, permitiendo una arquitectura modular y adaptable a los distintos componentes electrónicos del robot.
